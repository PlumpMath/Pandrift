# Pandrift

Oculus Rift code for Panda3D

[source @ github](https://github.com/wamonite/Pandrift)

## Usage

The example application displays the Panda3D tutorial scene.

* f - Toggle fullscreen
* r - Toggle Rift display
* Escape - quit
